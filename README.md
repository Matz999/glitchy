# Glitchy

Simple, local web app for applying glitchy image effects to photos.

## Usage

1. Open the project in a modern browser:
   - Recommended: serve locally to avoid module/file restrictions.
     - With Node: `npx http-server . -p 8080`
     - With Python: `python -m http.server 8000`
   - Then open: `http://localhost:8080`

2. Upload an image using the "Choose Image" button.
3. Click any effect button to apply it.
4. Adjust the "Datamosh block size" slider to change datamosh block size.
5. Click Download to save the canvas as PNG.
